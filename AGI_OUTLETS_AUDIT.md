## Access Method Summary

### ✅ RSS Available (~105 outlets)
Most outlets support standard RSS. Key AI-specific RSS paths:
- TechCrunch AI: `/category/artificial-intelligence/feed/`
- The Verge AI: `/ai-artificial-intelligence/rss/index.xml`
- The Guardian AI: `/technology/artificialintelligenceai/rss`
- ZDNet AI: `/topic/artificial-intelligence/rss.xml`
- arXiv: separate feeds for `cs.AI`, `cs.LG`, `cs.CL`, `cs.CV`

### 🔌 APIs Available (12 outlets)
| Source | API Type | Notes |
|--------|----------|-------|
| arXiv | arXiv API | Free, powerful, bulk paper access |
| Semantic Scholar | S2 API | Free, citation graph + paper data |
| Papers With Code | REST API | SOTA benchmarks, paper+code links |
| Hugging Face | HF Hub API | Model releases, dataset uploads |
| The Guardian | Open Platform API | Requires free API key |
| Reuters | Reuters Connect | Licensed/enterprise only |
| AP | AP Content API | Licensed only |
| Bloomberg | Bloomberg Data API | Enterprise, expensive |
| Medium | Medium API | For TDS and other Medium pubs |
| Substack | Unofficial RSS/API | RSS reliable; no official write API |
| BeehiIV | BeehiIV API | For Ben's Bites, Superhuman AI etc |
| LessWrong/AF | GraphQL API | Free, no key required |

### 🔒 Paywalled Outlets (Require Special Access)
- **Bloomberg Technology** — hard paywall; RSS limited
- **Wall Street Journal Tech** — Dow Jones credentials
- **Financial Times** — FT.com subscription
- **The Information** — premium subscription (~$399/yr)
- **Stratechery** — partial paywall; free posts available

### 🕷️ Require Scraping (No RSS/API)
- Stanford AI Index — annual PDF publication

## Implementation Notes for Monitoring System

### RSS Ingestion Priority
```
Priority 1 (multiple/day): techcrunch, wired, the_verge, venturebeat, reuters, bens_bites, tldr_ai
Priority 2 (daily): mit_tech_review, ars_technica, zdnet, bbc_tech, ai_news, arxiv_ai
Priority 3 (weekly): import_ai, the_batch, exponential_view, stanford_hai, alignment_forum
Priority 4 (monthly+): miri, nature_ai, partnership_on_ai, brookings_ai
```

### Deduplication Logic
- Bloomberg + Reuters + AP often share the same news; use title fingerprinting
- ArXiv preprints often covered 24-48h later by TechCrunch/MIT TR; cluster by paper ID
- Corporate blog announcements usually cause same-day coverage across Tier 1

### Quality Scoring Signals
- Tier 1 + Tier 3 institutional sources = highest credibility weight
- Multiple independent outlets covering same story = high-confidence signal
- Corporate official sources = flag as "primary source" not "independent coverage"
- Critical/skeptical outlets covering same story as optimist outlets = genuine debate signal
